# 실습 A: 이력서 → 포트폴리오 웹사이트

아래 내용을 채운 후, Antigravity에 통째로 붙여넣으세요.

---

## 1. 내 정보

```
이름:
직함: (예: UI/UX 디자이너, 그래픽 디자이너, 브랜드 디자이너)
한 줄 소개: (예: 사용자 중심의 인터페이스를 설계합니다)
이메일:
링크: (포트폴리오, 비핸스, 인스타그램 등)
```

## 2. 경력/학력 (최근순 3개)

```
1. 회사/학교명 | 직책/전공 | 기간
2.
3.
```

## 3. 대표 프로젝트 (3개)

```
프로젝트 1
- 이름:
- 설명: (1~2문장)
- 내 역할:

프로젝트 2
- 이름:
- 설명:
- 내 역할:

프로젝트 3
- 이름:
- 설명:
- 내 역할:
```

## 4. 스킬

```
주요 도구: (예: Figma, Photoshop, Illustrator)
기술 스택: (예: HTML/CSS, Framer, Webflow)
언어: (예: 한국어, 영어)
```

---

## Antigravity에 보낼 프롬프트

위 내용을 채운 후, 아래 프롬프트와 함께 붙여넣으세요.

```
아래 이력서 내용으로 1페이지 포트폴리오 웹사이트를 만들어줘.

[구조]
- Hero: 이름 + 직함 + 한 줄 소개
- About: 경력/학력
- Work: 프로젝트 3개 (카드 레이아웃)
- Skills: 도구 + 기술
- Contact: 이메일 + 링크

[디자인]
- 배경: 어두운 색 (#0a0a0a)
- 강조색: 노란색 (#FFD600)
- 본문 폰트: Pretendard
- 제목 폰트: 큰 볼드체
- 깔끔하고 미니멀한 스타일
- 스크롤 시 섹션마다 페이드인 애니메이션

[여기에 위에서 채운 내 정보를 붙여넣기]
```

---

## 수정 프롬프트 예시

만든 후 수정할 때 이렇게 말하세요:

| 원하는 변경 | 프롬프트 |
|------------|---------|
| 색상 변경 | "강조색을 파란색(#3B82F6)으로 바꿔줘" |
| 폰트 변경 | "제목을 세리프체로 바꿔줘" |
| 섹션 추가 | "프로젝트 섹션 아래에 수상 경력 섹션 추가해줘" |
| 레이아웃 변경 | "프로젝트 카드를 2열 그리드로 바꿔줘" |
| 다크/라이트 | "밝은 배경 버전으로 바꿔줘" |
| 반응형 | "모바일에서 잘 보이게 해줘" |
| 애니메이션 | "호버하면 카드가 살짝 올라오게 해줘" |
